# my-scratch-project

本项目包含一个 Scratch 游戏项目和一个 Ruby 脚本。

## 项目内容
- `whack-a-mole.sb3`：一个“打地鼠”类型的 Scratch 游戏项目文件。
- `generate.rb`：用于生成或处理 Scratch 项目的 Ruby 脚本。

## 操作步骤

1. 打开 `whack-a-mole.sb3` 文件：
   - 使用 Scratch 官网（https://scratch.mit.edu/）或本地 Scratch 编辑器导入该文件，即可体验和编辑游戏。

2. 运行 Ruby 脚本（可选）：
   - 确保已安装 Ruby 环境。
   - 在终端进入项目目录，运行：
     ```bash
     ruby generate.rb
     ```
   - 按脚本说明进行操作。

如有疑问请联系项目作者。
